Quick start guide for this project

- Download and install Python 3.7
- Open Visual Studio Code, open the folder "01 tutorial code"
- Open textract_table.py (as an example)
- Open a terminal from VSCode
- Type "py -3 -m venv env"  (create a virtual environment, a jail)
- Type ".\env\Scripts\activate" (enters virtual environment)
- Type "pip install -r .\requirements.txt" (installs necessary libraries)
- Press F5 to run the code
